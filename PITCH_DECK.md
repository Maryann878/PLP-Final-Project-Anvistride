## Presentation Tips

### For Live Presentations

1. **Slide Timing**: 2-3 minutes per slide
2. **Total Duration**: 15-20 minutes + Q&A
3. **Key Focus**: Problem, Solution, SDG Alignment, Demo
4. **Demo**: Show live application if possible
5. **Visuals**: Use screenshots and diagrams
6. **Engagement**: Ask questions, tell stories

### For Investor Pitch

- Emphasize: Market opportunity, traction, team
- Highlight: Competitive advantage, scalability
- Show: Live demo, user testimonials (future)
- Address: Business model, revenue potential

### For Academic/PLP Presentation

- Emphasize: SDG alignment, learning journey
- Highlight: Technical achievements, challenges overcome
- Show: Code quality, documentation, deployment
- Address: Skills learned, impact created
